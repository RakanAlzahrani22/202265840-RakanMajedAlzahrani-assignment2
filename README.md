# Assignment 2 – Interactive Features

This project is a simple responsive personal portfolio website built using HTML, CSS, and JavaScript. It was created as part of Assignment 1 to demonstrate foundational web development skills and responsible AI-assisted development.

---

## 👨‍💻 About the Project

The website includes:

- An **About Me** section introducing who I am and my academic background.
- A **Projects** section showcasing two JavaFX applications:
  - FaceLite Application
  - Resource Reserving System
- A **Contact** section with a responsive form (frontend only).
- Basic JavaScript interactivity.
- Responsive design for desktop, tablet, and mobile screens.

The goal of this assignment was to practice clean structure, responsive layout, and simple interactivity.

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- JavaFX (for showcased projects)
- Git & GitHub

AI tools such as ChatGPT were used during development to:

- Improve CSS styling
